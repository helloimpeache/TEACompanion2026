# 📱 TEACompanion

Aplicativo para auxiliar pais no monitoramento da rotina e comportamento de crianças com TEA, permitindo o compartilhamento com profissionais da saúde.

---

## 🚀 Funcionalidades

* Cadastro de usuários
* Gerenciamento de rotina
* Registro de comportamento
* Dashboard com análise
* Compartilhamento com psicólogos

---

## 🧠 Tecnologias

* Backend: Laravel
* Banco de Dados: MySQL
* Metodologia: Scrum / Scrumban
